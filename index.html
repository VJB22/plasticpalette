Project

Plastic Palette

Plastic Palette is a unique collection of 480 dominant colors from 80 photos that correspond to the search query "plastic waste".

Material

Images from Pexels.com were used for this project since they are free high-quality photos that anybody can download and reuse.

Idea

The idea behind this project was to scrape pictures from Pexels.com using Python libraries and then assemble them into a montage on Pixlr.com.
